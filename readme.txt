== Installation ==

1. Upload all the files into your wp-content/plugins directory, be sure to put them into a folder called "wp-broadcaster"
2. Activate the plugin at the plugin administration page
3. Add new credentials and set up broadcast
4. Run or schedule broadcasts

Please see the [WP-Broadcaster plugin home page](https://usabilitydynamics.com/products/wp-broadcaster/) for details.
